gcpbookmarklet by petercamfield

Bookmarklet for Google Cloud Print
Instructions

On a Windows / OS X machine

Google Chrome > Click on Spanner > Preferences > Under the Hood > Google Cloud Print > Enable

On a device you wish to print from

Add the following as a bookmark [to the bookmarks bar for easy access]: Add Print Button

Tap on the bookmark on any page you wish to print. Scroll to the bottom and tap on the print button!

If you are using an iPad/iPhone adding bookmarks is not very easy. You'll need to follow the instructions below. The good news; it's a one time set up.
Alternatively you can use the synching features of iCloud to sync your bookmarks from your Mac.

  1. Tap the action arrow on the toolbar in Safari. It's next to the address entry box at the top of the page
  2. Tap Add Bookmark
  3. Edit the title - I suggest something along the lines of 'Add Print Button'
  4. Tap Save
  5. Select and copy the Javascript below. Note - holding your finger down will highlight the black area only. Once this is done tap Copy - you'll get all of the text!
  6. Tap the bookmark icon on the toolbar
  7. Tap edit. Select the bookmark you created above and paste the location
  8. Tap Bookmarks and then tap Done
Javascript to copy: 
javascript:(function(){var script=document.createElement('SCRIPT');script.src='http://petercamfield.github.com/gcpbookmarklet/gcpbookmarklet.js';script.type='text/javascript';document.body.appendChild(script);})()

Enjoy!
